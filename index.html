Backup

Everything is stored only on this device. An automatic backup runs quietly in the background too, kept completely separate from your main data.

Only use this if the app is stuck (e.g. a database error banner). This permanently erases everything stored on this device (except the automatic backup) — export a manual backup first if you can.

Undo a deletion

Every time you delete something, a full snapshot is saved from right before it happened — restoring one puts everything back to that exact moment (the last 20 deletions are kept).
